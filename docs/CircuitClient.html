    CircuitClient

    the client which is used to make circuit requests to gridtime. Basically we will use this class to fire an event which the main process listens for. On notification it will make a rest call to grid time. the response is the piped into the calling function to this client. EXAMPLE: `CircuitClient.createLearningCircuitModel("angry_teachers", this, model => { console.log(model); });`
